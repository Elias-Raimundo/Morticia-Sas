# Morticia-SAS

Sistema web para gestión de pedidos y clientes.

Permite a los clientes crear pedidos y a los administradores gestionar productos, clientes, saldos y estadísticas.

---

#   Autor

Elias Raimundo

# Tecnologías utilizadas

Frontend
- Next.js
- React
- TailwindCSS

Backend
- Node.js
- Express

Base de datos
- PostgreSQL
- Prisma ORM
- Supabase

Infraestructura
- Frontend: Vercel
- Backend: Render
- Base de datos: Supabase

---

# Funcionalidades

Clientes
- Registro de usuarios
- Inicio de sesión
- Crear pedidos
- Historial de pedidos
- Visualizar saldos

Administración
- Gestión de productos
- Balance de clientes
- Registro de pagos
- Visualización de pedidos
- Estadísticas de ventas
- Productos más vendidos

Sistema
- Generación de pedidos en PDF
- Aplicación instalable en celular (PWA)
- Interfaz responsive para móvil
